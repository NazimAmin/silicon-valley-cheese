## Install

```
$ npm install -g https://github.com/NazimAmin/silicon-valley-cheese 
$ cheese-bae
```

## Launch in terminal at startup 

```
$ npm install -g https://github.com/NazimAmin/silicon-valley-cheese && echo "cheese-bae" >> ~/.bashrc
```
